1. Start Test Network
```console
./network.sh up
```

2. Command to create the channel
```console
./network.sh createChannel
```

3. Command to stop the fabric network
```console
./network.sh down
```

4. Command to deploy the chaincode
```console
./network.sh deployCC -ccn regnet -ccl javascript -ccv 1 -ccs 1 -ccp ../chaincode -cci instantiate
```
